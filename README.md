# BlinkIT Sstore Analysis

## Project Overview
This project analyzes BlinkIT sales data using SQL and Power BI. The objective is to identify sales trends, customer preferences, and outlet performance to generate actionable business insights.

## Tools Used
- PostgreSQL
- Power BI
- Excel
- GitHub

## Data Cleaning
- Removed duplicates
- Handled missing values
- Corrected data types
- Standardized column names

## SQL Analysis
Key business questions answered:
1. Total Sales
2. Average Sales
3. Sales by Outlet Type
4. Sales by Item Type
5. Top Performing Outlets
6. Average Rating by Outlet
7. Sales by Fat Content
8. Outlet-wise Revenue
9. Top Selling Categories
10. Year-wise Outlet Analysis

## Power BI Dashboard
Dashboard includes:
- KPI Cards
- Sales Trend Analysis
- Outlet Performance
- Item Category Analysis
- Interactive Filters

## Key Insights
- Supermarket Type1 generated the highest sales.
- Fruits and Snacks contributed significantly to revenue.
- Medium-sized outlets performed better than small outlets.
- Higher-rated products generally achieved better sales.
